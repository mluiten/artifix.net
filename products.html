Android

Pomodroido: time-boxing with a side of achievement

An android app that does time-boxing using The Pomodoro Technique, but adds some game mechanics. Also syncs to profile.pomodroido.com to provide some better analytics.

Website

Open-Eyes

Online questionairre tool made for NL-Eyes. Iterated through 4 major versions from 2005 to 2011. It is mostly used to perform customer- and employee satisfaction surveys in childcare centres. The latest version is based on Zend Framework using Doctrine 2 and MongoDB. Currently working on subscription-based access to open it to the public.

Website

TrendingOnTV.com

Watches for tweets of all the shows currently on the Dutch television and ranks them by popularity. An experimental website that turned out pretty nice. Uses Node.JS and the Twitter API.

Website

NUWIJ.org

Watches for tweets with a certain hashtag (#nuwij) and uses websockets to push updates to the client. Uses Node.JS, socket.io and the Twitter API.

SSR-Rooster.nl

Schedules all the rosters for sorority SSR-Leiden, and sends text messages (through Mollie) as reminders. Access limited, but screenshots will be posted soon.
